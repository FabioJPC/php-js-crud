## Code academy project 3 - Http

### Instalation instructions

### Clone the project

```bash
git clone https://github.com/FabioJPC/php-js-crud.git
cd php-js-crud
```

### Install frontend dependencies and build assets
```bash
cd frontend
npm install
npm run build
```

### Move to root directory and start all containers
```bash
cd ..
docker compose up -d --build
```

### Test the application at http://localhost:5000
